# Blog Website

A full-stack blog website built to allow users to create, read, edit, and manage blog posts through a modern web interface.

## Tech Stack

React.js
Node.js
Express.js
MongoDB
JavaScript
 HTML & CSS

##Features

Create blog posts
Read blog posts
Edit existing posts
Delete posts
Responsive user interface
